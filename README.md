# file-relay-hub

Node.js + TypeScript + Fastify service skeleton.

## Requirements

- Node.js 20+
- npm 10+

## Setup

```bash
npm install
```

## Run in development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Start (after build)

```bash
npm run start
```

## Health Check

- `GET /health`
